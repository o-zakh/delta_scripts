Установка
  1. Установи расширение Tampermonkey из Chrome Web Store.
    1. Нажми ПКМ на установленное расширение -> Управление расширениями
    2. Включи "Разрешить пользовательские скрипты"
  2. Попроси ссылку на тестовую версию скрипта у o.zakharov 👋
  3. На открывшейся вкладке нажми "Установить"

После установки скрипт может время от времени запрашивать обновления по мере их выхода

Как пользоваться:

  1. Перейди в админку основного аккаунта мобильного игрока.
  2. На странице появится кнопка "Верификация переноса" и кнопка настроек "⚙️":
  3. Нажми "Верификация переноса".
  4. Введи ID основного аккаунта (предзаполнено из URL) и ID нового аккаунта.
  5. Не переключайся на другие вкладки, пока скрипт собирает информацию.
  6. Ответь на выскочившие попапы.
  7. Результат скопируется в буфер обмена.
  8. Просто сделай Ctrl+V во внутреннее примечание тикета.

С помощью кнопки "⚙️" можно настраивать ID сплитов для проверки (по умолчанию: 380,391,425,431). Эта информация локально сохраняется в вашем браузере, поэтому менять её можно редко и по необходимости
